========================================================
   ENERGY CONSUMPTION PREDICTION - PROJECT REPORT
========================================================

1. EXECUTIVE SUMMARY
---------------------
The goal of this project was to predict appliance energy consumption using environmental sensor data. After testing multiple machine learning algorithms, the **Random Forest Regressor** was identified as the most accurate and reliable model for this task.

WINNER: RANDOM FOREST REGRESSOR ⭐
- Best Error Reduction (MAE: 33.70)
- Highest Variance Explained (R2: 0.53)

2. DATASET OVERVIEW
--------------------
- Source: UCI Appliances Energy Prediction Dataset
- Scope: 19,735 data points tracking 4.5 months of sensor metrics.
- Key Variables: Indoor/Outdoor Temperature, Humidity, and Weather Data.

3. PERFORMANCE METRICS (COMPREHENSIVE)
---------------------------------------

A. REGRESSION MODELS (Numerical Prediction)
   
   | Algorithm           | MAE (Error)  | R2 (Accuracy) |
   |---------------------|--------------|---------------|
   | Random Forest       | 33.70        | 0.53          |
   | Decision Tree       | 40.02        | 0.11          |
   | Linear Regression   | 53.19        | 0.16          |

   *Note: MAE (Mean Absolute Error) - lower is better. R2 (R-squared) - higher is better.*

B. CLASSIFICATION MODELS (Categorical Prediction)

   | Algorithm           | Metric       | Result        |
   |---------------------|--------------|---------------|
   | Logistic Regression | Accuracy     | 80.39%        |

   *Task: Predicting if energy usage will be "High" or "Low" relative to the average.*

4. ALGORITHM INSIGHTS
----------------------
- Random Forest: Most robust. By combining 100+ decision trees, it captures complex patterns that single models miss.
- Decision Tree: Fast but prone to instability (high variance).
- Linear Regression: Simple but struggled with the non-linear nature of residential energy usage.
- Logistic Regression: Excellent for quick binary classification of consumption levels.

5. CONCLUSION
--------------
This project successfully demonstrates that ensemble methods like Random Forest are superior for predicting high-frequency sensor data like energy consumption. The final results provide a reliable baseline for building predictive energy-saving systems.
